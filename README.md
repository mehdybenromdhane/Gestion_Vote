## Quick start

front:
1- `npm install`
2- `npm start`

back:
1- `cd server`
2- `npm install`
3- `npm run server`

## Main Fonctions:

1- Add subject by the user after login.
2- Display all subjects
3- vote yes or no.
4- see percentage

## Technologies:

1- react js
2- node js
3- mongo db
4- Exrpess js
